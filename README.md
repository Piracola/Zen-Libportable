# Zen-Libportable
本项目旨在提供一个开箱即用的便携版 Zen 浏览器。
基于 libportable 的浏览器便携版自动化打包项目，支持 Zen 浏览器和 [Firefox](https://github.com/Piracola/Firefox-Libportable) 浏览器。

## 使用方式
1. 下载 [Zen.zip](https://github.com/Piracola/Zen-Libportable/releases/latest/download/Zen.zip) 文件
2. 解压到任意目录
3. 运行 "开始.bat" 即可创建快捷方式

## 更新说明
仅需替换 Zen 这个包含浏览器本体的目录即可升级。
1. 在浏览器根目录中将 Zen 目录修改为 Firefoxold 以便于回退。
2. 复制新版本压缩包内的 Zen 到同目录下。

## 参数配置
配置文件目录修改：
- 参考[portable](libportable/portable.ini)文件中的说明对便携化参数进行自定义。
